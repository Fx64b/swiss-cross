# Contributing to Swiss Cross

Thank you for your interest in contributing to Swiss Cross!

## Development Setup

1. Fork and clone the repository
2. Install dependencies: `pnpm install`
3. Start development mode: `pnpm dev`

## Making Changes

1. Create a new branch for your feature/fix
2. Make your changes
3. Add tests for new functionality
4. Run tests: `pnpm test`
5. Run build: `pnpm build`
6. Ensure all tests pass and build succeeds

## Code Style

- Follow existing TypeScript patterns
- Use meaningful variable names
- Keep functions small and focused
- Add JSDoc comments for public APIs

## Testing

- Write tests for all new features
- Ensure existing tests continue to pass
- Aim for good test coverage

## Pull Request Process

1. Update the README if needed
2. Add tests for your changes
3. Ensure CI passes
4. Request review from maintainers

## Questions?

Feel free to open an issue for any questions about contributing.
